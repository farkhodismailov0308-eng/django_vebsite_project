# Django vebsite project

## O'rnatish

```bash
git clone https://github.com/farkhodismailov0308-eng/django_vebsite_project.git
cd django_vebsite_project
python -m venv myenv
source myenv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
